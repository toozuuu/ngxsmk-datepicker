# Storybook Configuration

This directory contains the Storybook configuration for the ngxsmk-datepicker component library.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start Storybook:
   ```bash
   npm run storybook
   ```

3. Build static Storybook:
   ```bash
   npm run build-storybook
   ```

## Configuration Files

- `main.ts` - Main Storybook configuration
- `preview.ts` - Preview configuration and global decorators
- `manager.ts` - Storybook manager UI configuration

## Stories

Stories are located in `projects/ngxsmk-datepicker/src/lib/ngxsmk-datepicker.stories.ts`

Each story demonstrates different features and use cases of the datepicker component with interactive controls.
