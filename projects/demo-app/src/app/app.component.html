API Reference

Inputs

Property Type Default Description
mode 'single' | 'range' | 'multiple' 'single' Selection mode
value DatepickerValue null Programmatic value setting
placeholder string 'Select Date' Input placeholder text
minDate DateInput | null null Minimum selectable date
maxDate DateInput | null null Maximum selectable date
showTime boolean false Show time selection
inline boolean | 'always' | 'auto' false Inline calendar display
theme 'light' | 'dark' 'light' Theme variant

Outputs

Event Type Description
valueChange EventEmitter<DatepickerValue> Emitted when value changes
action EventEmitter<{ type: string; payload?: any }> Emitted on user actions

Examples

Single Date Selection

Basic single date picker with form control integration.

html
Form Value: {{ JSON.stringify(datepickerForm.controls.singleDate.value) }}

Date Range Selection

Select a date range with optional time selection.

html
Form Value: {{ JSON.stringify(datepickerForm.controls.rangeWithTime.value) }}

Multiple Date Selection

Select multiple non-contiguous dates.

html
Form Value: {{ JSON.stringify(datepickerForm.controls.multipleDates.value) }}

Programmatic Value Setting

Set the datepicker value programmatically, useful when data comes from a server-side API.

typescript
Current Value: {{ programmaticSingleDate ? (programmaticSingleDate | date:'medium') : 'Not set' }}
